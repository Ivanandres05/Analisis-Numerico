# Proyecto Métodos Numéricos

Resolución de ecuaciones no lineales aplicado a Ingeniería de Software.

## Instalación

1. Clona el repositorio/carpeta.
2. Es recomendado usar un entorno virtual.
3. Instala los requerimientos:
   ```bash
   pip install -r requirements.txt
   ```

## Ejecución

Corre el proyecto principal usando:

```bash
python main.py
```

## Pruebas

Para ejecutar las pruebas unitarias:

```bash
python -m unittest tests/test_metodos.py
```
